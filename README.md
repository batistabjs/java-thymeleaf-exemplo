# java-thymeleaf-exemplo
### Desenvolvimento:
Tecnologias: 
- Java 17
- Framework: Spring
- Gerenciador de Dependências: Maven
- Thymeleaf Engine

### Para execução do projeto:
Instalar o Java 17 e Maven. 

Na raiz do projeto, executar no terminal o comando:
```
mvn spring-boot:run
```

No navegador de internet, acesse:
```
http://localhost:8090
```
